Тег Описание
<a> Тег гиперссылки, который создает ссылку на другую веб-страницу, документ, изображение и т.д.
<img> Тег изображения, который добавляет на веб-страницу изображение.
<p> Тег абзаца, который создает новый абзац текста на веб-странице.
<ul> Тег неупорядоченного списка, который создает список элементов без порядкового номера.
<ol> Тег упорядоченного списка, который создает список элементов с порядковыми номерами.
<li> Тег элемента списка, который является элементом неупорядоченного или упорядоченного списка.
<div> Тег блока, который используется для группировки элементов на веб-странице.
<span> Тег строчного элемента, который используется для задания стиля для небольшой части текста на веб-странице.
<form> Тег формы, который используется для создания формы на веб-странице для отправки данных на сервер.
<input> Тег ввода, который используется для создания элементов формы для ввода текста, выбора из выпадающего списка и т.д.
<textarea> Тег многострочного текстового поля, который используется для ввода больших объемов текста на веб-странице.
<select> Тег выпадающего списка, который создает список элементов, из которых можно выбрать один.
<option> Тег элемента выпадающего списка, который определяет каждый элемент списка внутри тега <select>.
<button> Тег кнопки, который создает кнопку на веб-странице, которую пользователь может нажать для выполнения действия.
<label> Тег метки, который используется для связывания текста с элементом формы, таким как поле ввода.
<table> Тег таблицы, который используется для создания таблиц на веб-странице.
<tr> Тег строки таблицы, который определяет каждую строку таблицы.
<th> Тег ячейки заголовка таблицы, который определяет каждую ячейку заголовка таблицы.
<td> Тег ячейки таблицы, который определяет каждую ячейку таблицы, кроме ячеек заголовка таблицы.
<Style> Тег позволяет пользоватся функционалом CSS
<script> Тег позволяет пользоватся функционалом javascript
